# Advent of Code Solutions in Lua

This repository contains my personal solutions to the [Advent of Code](https://adventofcode.com/) challenges for every year from **2015 through 2025**, implemented in **Lua 5.4**.

---

## üìÅ Project Structure

Each year has its own directory, organized as follows:

```
/<year>/
   /day01/
      pt1.lua
      pt2.lua
      example-input.txt
      real-input.txt
   /day02/
      pt1.lua
      pt2.lua
      example-input.txt
      real-input.txt
  ...
```

**Example:**

```
/2015/day01/pt2.lua
/2021/day04/pt1.lua
```

- Each day typically has 2 files for part 1 and part 2
- Input files (if needed) are usually named `example-input.txt` and `real-input.txt` and placed in the respective `/dayXX/` folder.

---

## üõ†Ô∏è Requirements

- [Lua 5.4](https://www.lua.org/)

### Install Lua 5.4

**Linux/macOS:**

```
sudo apt-get install lua5.4        # Debian/Ubuntu
brew install lua                   # macOS/Homebrew
```

**Windows:**

- Download official binaries for [Lua 5.4](https://www.lua.org/ftp/) or use the [Lua for Windows](https://github.com/rjpcomputing/luaforwindows) distribution.

---

## ‚ñ∂Ô∏è Running Solutions

1. Navigate to the desired year/day directory:

   ```sh
   cd <year>/day<dd>
   ```

2. Run the solution with Lua (optionally passing input, depending on the script):

   ```sh
   lua solution.lua input.txt
   ```
